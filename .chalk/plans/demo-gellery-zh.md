# 演示画廊 —— 可下载的组件展示库

## 背景
当前模板系统（以抽屉形式展示原始 HTML 字符串）无法充分体现智能体生成界面的价值。我们将其替换为**精选的、已保存对话结果的画廊**——每个条目都是一个快照：**1 条用户指令 → 1 个生成组件**。这样访问者可以直观看到智能体的输出效果，并与真实生成内容交互。

关联 issues：#49、#55
依赖 issues：#14/#42（下载/复制为代码功能）提供导出工具。

---

## 画廊展示两大类别
### 1. 讲解与可视化类
教育/说明性生成组件，例如：
- “飞机飞行原理”交互式 3D 讲解
- 二分查找步骤可视化
- 太阳系模型
- 神经网络前向传播动画

用于展示：智能体可通过简单指令，生成丰富、可交互的教育类内容。

### 2. 自定义界面与风格化组件
符合特定设计语言或品牌风格的生成组件，例如：
- Clippy 风格助手界面
- Spotify 风格音乐仪表盘
- 特定设计系统下的发票卡片
- 主题化控制面板

用于展示：生成的界面并非千篇一律，智能体可以产出设计感强、风格明确的组件。

---

## 单条画廊条目包含内容
- 原始用户指令（单条消息）
- 生成的组件结果（可渲染、可交互）
- **下载**生成组件的功能（通过 #14/#42 的 `export-utils.ts` 实现）

## 布局 —— 参考 variant.com 风格
### 当前结构
主视图为聊天界面，右侧滑出模板抽屉。

### 新结构
左侧固定面板 + 右侧瀑布流卡片网格
```
+---------------------+-------------------------------------------+
| Logo                |                                           |
|                     |  [卡片]      [高卡片]      [卡片]         |
| 输入想法，生成无限  |  [宽卡片]        [卡片]                   |
| 界面，滚动浏览即可  |  [卡片]      [卡片]      [高卡片]         |
|                     |  [卡片]      [宽卡片]                     |
| 描述文本            |  [高卡片]    [卡片]      [卡片]           |
|                     |            … 无限滚动 …                   |
| [注册] [随机生成]    |                                           |
|                     |                                           |
| +-----------------+ |                                           |
| | 聊天输入框…     | |                                           |
| +-----------------+ |                                           |
+---------------------+-------------------------------------------+
```

### 核心布局规则
- **左侧面板**（约 340px，固定）：Logo、标题标语、描述、行动按钮（注册/登录、随机生成），底部固定聊天输入框。**面板不滚动**。
- **右侧区域**（占满剩余宽度）：瀑布流网格画廊卡片，可独立滚动。卡片根据内容宽高比自动调整尺寸。
- **卡片**：深色主题、圆角，通过 iframe 实时预览可交互的生成组件。鼠标悬浮时显示下载按钮与“立即尝试”按钮。
- **无分类筛选标签**：瀑布流布局本身即可体现内容多样性，讲解类与风格化组件交错排列。
- **聊天输入框**：位于左侧面板底部，而非独立抽屉。输入指令后生成新组件并加入网格。
- **深色主题**：整体页面使用深色背景，突出组件预览效果。

## 需要新建的文件

### 1. `apps/app/src/components/demo-gallery/gallery-data.ts`
画廊条目配置文件，每条为对话快照：1 条用户指令 → 1 个生成组件。

**接口定义：**
```ts
export interface GalleryItem {
  id: string;
  title: string;
  description: string;
  axis: "explainer" | "styled";  // 展示类别
  prompt: string;                // 原始用户指令
  html?: string;                 // 生成的 HTML（用于实时预览 + 下载）
  component_type?: string;
  component_data?: Record<string, unknown>;
  size?: "normal" | "tall" | "wide";  // 瀑布流尺寸
}
```

**精选条目（约 10 条，两类交错排列）：**

讲解与可视化类：
- 飞机飞行原理 —— 交互式 3D 讲解
- 二分查找 —— 步骤可视化
- 太阳系 —— 行星轨道运行
- 神经网络 —— 前向传播动画
- 排序算法对比 —— 冒泡排序 vs 快速排序

自定义界面与风格化组件：
- 天气仪表盘（复用现有示例 HTML）
- KPI 仪表盘（复用现有示例 HTML）
- 发票卡片（复用现有示例 HTML）
- 番茄钟计时器
- 电动车电池组件（参考 variant.com）

带有预渲染 `html` 的条目展示 iframe 实时预览；无 HTML 的条目展示带指令文本的风格化占位符。

## 依赖项
### 下载/导出功能（前置条件：#14/#42）
画廊卡片需要下载按钮。`export-utils.ts`（来自下载 PR）提供：
- `assembleStandaloneHtml`
- `triggerDownload`
- `slugify`

卡片悬浮层直接导入使用。

### 程序化发送消息
需确认 CopilotKit v2 API 支持程序化发消息：
1. `agent.sendMessage(text)`（如 v2 支持）
2. `agent.addMessage({ role: "user", content: text }) + agent.runAgent()`
3. 备选方案：程序化设置输入框值并触发提交事件
